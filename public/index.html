How the Auto Index Works
📡
Weekly Polls
Every 7 days we query Marketcheck's national active-listing database for each of the 25 models.
📊
Listing Medians
We collect the median listing price per model — a robust measure that resists skew from outlier vehicles.
🧮
Index Calculation
The AUTO INDEX equals the simple average of all 25 model medians — a broad, equally-weighted gauge.
🔍
Listing Prices Only
These are asking prices on active used-car listings — not transaction prices. Actual sale prices typically differ.
